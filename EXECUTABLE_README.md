# Smart Attendance System - Executable Version

## 📁 File Location
Your executable file is located at:
```
attendance_system\release\win-unpacked\Smart Attendance.exe
```

## 🚀 How to Run

### Option 1: Direct Execution
1. Navigate to the `attendance_system\release\win-unpacked\` folder
2. Double-click `Smart Attendance.exe`
3. The app will open in a desktop window

### Option 2: Use the Launcher (Recommended)
1. Double-click `Run_Attendance_System.bat` in the main folder
2. This will automatically start the application

## ⚠️ Important Notes

### First Time Setup
- **Camera Permissions**: The app will request camera access - click "Allow"
- **Antivirus**: Some antivirus software may flag the exe - add it to exceptions
- **Model Files**: Ensure all files in the `models/` folder are present

### System Requirements
- Windows 10/11
- Webcam/Camera
- At least 4GB RAM
- 2GB free disk space

### Troubleshooting
If the app doesn't start:
1. Check Windows Defender/Firewall settings
2. Try running as Administrator
3. Ensure no other applications are using the camera
4. Check if all model files exist

### Features Available
- ✅ Facial recognition attendance marking
- ✅ Real-time face detection
- ✅ Admin panel for student registration
- ✅ Attendance export to CSV
- ✅ Persistent data storage

## 📞 Support
If you encounter issues:
